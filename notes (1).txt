Paragraph:

    Definition: Paragraphs are used to group and structure blocks of text on a web page.
    Example code:   
    <p>This is a paragraph.</p>
    
    Paragraph:

    <p>: Used to create paragraphs of text.

    Paragraph:

Title:

    Definition: The title element defines the title of a document, which is displayed in the browser's title bar or tab.
    Example code:
    <head>
        <title>Page Title</title>
      </head>

      Title:

      <title>: Specifies the title of the HTML document.  

      Title:

       No specific attributes.

 Marquee:

    Definition: The marquee element creates a scrolling or moving effect for text or images.
    Example code:

      <marquee>This is scrolling text.</marquee>
    
    Marquee:

     <marquee>: Creates a scrolling or moving effect for text or images.
            
    Marquee:

    behavior: Specifies the scrolling behavior ("scroll", "slide", or "alternate").
    direction: Specifies the direction of the scrolling ("left", "right", "up", or "down").
    loop: Specifies the number of times the marquee should loop.
    scrollamount: Specifies the speed of the scrolling.
    bgcolor: Specifies the background color of the marquee.

Image:

    Definition: The image element is used to insert images into an HTML document.
    Example code:

    <img src="image.jpg" alt="Description of the image">
    
    Image:

    <img>: Inserts images into an HTML document.

     Image:

     src: Specifies the URL or file path of the image.
     alt: Specifies alternative text for the image (displayed if the image cannot be loaded).
     width: Specifies the width of the image.
     height: Specifies the height of the image.

List:

     Definition: Lists are used to present information in an organized manner, either as an ordered list (numbered) or an unordered list (bullet points).
     Example code:
     <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
      </ul>
 
      List:

      <ul>: Creates an unordered list (bullet points).
      <ol>: Creates an ordered list (numbered).
      <li>: Defines a list item

      List:

      No specific attributes.

Link:

      Definition: Links are used to navigate between different web pages. They are created using the anchor element.
      Example code:

      <a href="https://www.example.com">Visit Example Website</a>
      
      Link:

      <a>: Creates a hyperlink to another web page or resource.

      Link:

      href: Specifies the URL of the linked page or resource.
      target: Specifies where to open the linked page ("_blank", "_self", "_parent", "_top", or a custom frame name).


Table:

      Definition: Tables are used to organize data into rows and columns.
      Example code:
      <table>
        <tr>
          <th>Header 1</th>
          <th>Header 2</th>
        </tr>
        <tr>
          <td>Data 1</td>
          <td>Data 2</td>
        </tr>
      </table>
      
      Table:

      <table>: Creates a table.
      <tr>: Defines a table row.
      <th>: Defines a table header cell.
      <td>: Defines a table data cell.
      <caption>: Adds a caption to the table.
      <thead>, <tbody>, <tfoot>: Divides the table into different sections (header, body, footer).

      Table:

      border: Specifies the border width of the table.
      width: Specifies the width of the table.
      height: Specifies the height of the table.
      cellpadding: Specifies the space between the cell content and cell border.
      cellspacing: Specifies the space between cells.
      align: Specifies the alignment of the table ("left", "center", "right").
      bgcolor: Specifies the background color of the table.
